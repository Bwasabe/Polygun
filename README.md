# 폴리건 Polygun

미궁을 탐험해 소 괴물을 잡고 애인을 구출하자!

폴리건은 3인칭 액션게임으로, 구 모양의 주인공의 애인을 갑자기 나타난 소 괴물에 빼앗아 버리게 되고,
그 애인을 구출하기 위해 던전에 들어가게 되는 3D 액션 게임 입니다.

여기서 저는 플레이어와 아이템들, 보스 등을 담당하였습니다


<b>조작 방법</b> :
  - 이동 : W A S D
  - 점프 : Space
  - 대쉬 : LShift
  - 능력 사용하기 : Q
  - 일시 정지 : ESC
  - 상호작용 : E
  
  ※ <b>주의!</b> 마우스 감도가 빠름
